# NebulaCalc Pro - Advanced Scientific Calculator

NebulaCalc Pro is a highly precise, professional-grade scientific calculator web application. It combines a stunning futuristic glassmorphic interface with a powerful mathematical evaluation engine, delivering exceptional accuracy and an extensive range of scientific functions.

This project demonstrates advanced DOM manipulation, event delegation, secure expression parsing, state management, and modern CSS design techniques.

## Features

### Advanced Scientific Engine

* Full support for scientific operations including:

  * Trigonometric functions (`sin`, `cos`, `tan`)
  * Factorials (`!`)
  * Modulo (`%`)
  * Square roots (`√`)
  * Powers (`^`)
  * Parentheses and proper order of operations

### High-Precision Mathematics

* Powered by **math.js** for reliable and accurate calculations.
* Eliminates native JavaScript floating-point precision issues.
* Example:

  * `0.1 + 0.2 = 0.3`

### Dynamic History Log

* Automatically stores calculation history using `localStorage`.
* Scrollable history panel for quick access to previous calculations.
* Click any history entry to instantly load the result back into the display.

### Professional Event Delegation

* Utilizes a single event listener on the calculator grid.
* Improves performance and maintainability.
* Demonstrates efficient JavaScript event handling practices.

### Comprehensive Keyboard Support

* Operate the calculator entirely using your physical keyboard.
* Fast and intuitive user experience.

### Premium Glassmorphism Interface

* Custom SVG noise/grain overlay.
* Animated multi-color mesh gradient background.
* Deep inset shadows and layered visual effects.
* Modern, tactile user interface.

### Dynamic Theme Switching

* Instantly switch between:

  * Deep Space Dark Mode
  * Frosted Glass Light Mode
